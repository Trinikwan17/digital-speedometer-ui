# ⌨️ Digital- Speedometer

A React-based Typing Speed Tester application that calculates a user’s typing speed in real-time.  
The project measures Words Per Minute (WPM), accuracy, and time taken while typing a given text.

---

## 📌 Features

- ⏱ Real-time typing timer
- 🚀 Calculates Words Per Minute (WPM)
- 🎯 Accuracy percentage calculation
- 🔄 Reset functionality
- 📱 Responsive UI
- ⚛️ Built using React + Vite
- 🌙 Dark Mode toggle feature

---

## 🛠 Tech Stack

- React
- Vite
- JavaScript
- CSS3

---

## 🧠 How It Works

- User starts typing the given paragraph.
- Application calculates:
  - Total characters typed
  - Correct characters
  - Words per minute (WPM)
  - Accuracy %
- Final result is displayed once time ends.

## 🚀 Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/Trinikwan17/digital-speedometer-ui.git
```

2️⃣ Navigate into project

```
cd digital-speedometer-ui/my-app
```

3️⃣ Install dependencies

```
npm install
```

4️⃣ Start development server

```
npm run dev
```
